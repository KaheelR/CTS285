# TIC TAC TOE

This is a Tic Tac Toe game made on streamlit, this game allows you to play with another person
It also allows you to play against an CPU with a easy and hard difficulty.


To run this application you must first download the folder then run these set of commands in a terminal.  
cd tic-tac-toe  
python -m venv venv  
source venv/bin/activate  # On Windows: venv\Scripts\activate  
pip install streamlit  
streamlit run app.py
